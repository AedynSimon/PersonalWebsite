# Aedyn Simon Portfolio Website

This repository contains the source code for my personal portfolio website built using **HTML5**, **CSS3**, and **JavaScript**. The website serves as a showcase of my work, skills, and experience in programming.

---

## Features
- **Home**: A beautiful home page that has my downloadable resume.
- **Programming Section**: Highlights key projects I’ve worked on while also showcasing my skills in programming and software development.
- **About Section**: Provides a brief introduction about me, my background, and interests.
- **Contact Section**: Allows visitors to get in touch with me through links to my email and social media profiles.
- **Responsive Design**: Optimized for viewing on devices of all sizes.

---

## Technologies Used
- **HTML5**: For the website's structure and content.
- **CSS3**: For styling and layout, ensuring a clean and modern design.
- **JavaScript**: For adding interactivity and enhancing the user experience.
